# 尽调报告 — 结构模板

这是报告的固定骨架，改编自银行系AIC（金融资产投资公司）股权投资试点业务真实尽调报告样本。每次生成都遵循这个结构；标题文字可按具体交易微调（比如基金名称、标的公司），但部分顺序、层级和覆盖范围不变。样本报告标题格式为"关于为XX公司办理不超过X亿元股权投资试点业务的尽调报告"，落款为提交日期。

## 篇幅基准表（按参照样本实测字数，含表格文本）

样本全文约 **5万字**（含表格），纯分析性正文约3万+字。下表是各部分/子章节的实测字符数，写作时用它校准深度——目标是"同一量级"，不是精确凑字数。字数差距的来源必须是**颗粒度**（多写几个科目、几个人、几家竞品、几条风险子项），不能靠空话或重复同一事实凑篇幅。

| 章节 | 样本实测字数 | 篇幅的来源（复刻这个，而不是复刻字数本身） |
|---|---|---|
| 声明与保证页 | ~300字 | 固定措辞+签署行，本来就短 |
| **第一部分 项目方案（合计）** | **~3,000字** | |
| 　一、项目背景 | ~400字 | 一段电梯陈述 |
| 　二、尽调实施情况 | ~500字 | 逐条列出实际做过的每一项尽调动作 |
| 　方案概述+基本交易要素 | ~400-600字 | 9项交易要素逐条写 |
| 　基金方案 | ~1,300字 | 12项要素（名称/形式/注册地/规模/存续期/GP/LP/投资方向/决策机制/费率/退出/分配瀑布/重大事项）逐条展开，不是一段话带过 |
| 　融资安排及上市计划 | ~500-700字 | 保障条款逐项列出+回购触发情形逐条列出 |
| **第二部分 标的企业情况（合计）** | **~25,000字** | 全文篇幅最大的部分 |
| 　基本情况-公司简介 | ~4,800字 | 信息表+股权结构表+历次融资表，每轮融资、每个股东都要出现 |
| 　公司治理结构 | ~3,000字 | 治理层级+组织架构+**核心团队逐人成段介绍**（样本13人，每人100-200字，含教育背景+过往任职+可验证成就） |
| 　子公司及参股公司情况 | ~700-800字 | 子公司表 |
| 　二、企业经营情况 | ~4,700字 | 产品逐代对比+近期商业化进展逐条+客户结构 |
| 　三、财务情况 | **~9,700字** | 全文最密的一节：资产负债表每个科目一句变动说明、利润表每个科目一句、现金流三类分别说明、分产品分客户毛利明细表——**逐科目、逐客户展开，不是"财务状况良好"一句话带过** |
| 　四、资信情况 | ~1,900字 | 天然较短，多为查询结果确认，不用刻意拉长 |
| **第三部分 行业分析与竞争优势（合计）** | **~6,000字** | |
| 　一、行业技术发展情况 | ~4,300字 | 背景/原理/分类/介质选择/趋势/应用领域/挑战 **七个子标题，每个都要有独立成段的论述**，不能几句话带过整节 |
| 　二、公司竞争优势 | ~1,600字 | 5条优势逐条成段论证，每条都要有具体依据 |
| **第四部分 同业可比分析（合计）** | **~7,500字** | 每家竞品公司都要有完整段落画像（成立时间/地点/技术路线/产品进展/收入/融资历史），不是一行提及；加多张参数对比表 |
| **第五部分 投资收益情况分析（合计）** | ~2,300字 | 盈利预测归因说明+可比公司估值表+至少两组情景假设，每组都写出假设前提 |
| **第六部分 上市退出与风险分析（合计）** | ~5,000字 | 每条编号风险都要有完整描述段+完整"应对措施"段，不能压缩成一行 |
| 第七部分 调查结论性意见 | ~350-400字 | 有意保持简短、信息密度高，不用拉长 |

**用法**：起草每一部分后，粗略估一下字数（中文字符数，不含表格可以用 `wc -m` 或直接数），和上表对比。明显偏短的，先检查是不是漏掉了应有的颗粒度（少写了几个科目/几个人/几家竞品/几条风险），把这些补全，而不是加形容词或空话凑字数。资信情况、结论性意见这两节天然短，不用往上凑。

## 声明与保证 / 签署页（报告首页，无需单独封面或目录）

固定包含：
- 报告标题（两行）："关于为XX公司办理不超过X亿元股权投资试点业务的尽调报告"
- 声明段落（固定措辞，据机构实际规章制度调整机构名）：已根据XX机构有关规章制度对纯股权项目进行调查，做到勤勉尽职；承诺不存在故意隐瞒负面消息、虚假记载，并对报告所涉及数据和资料的真实性负责。
- 签署行：尽职调查人（姓名+联系方式，可多人）、调查复核人、总经理/负责人、经营主责任人 — 姓名和联系方式必须由用户提供或来自项目组材料，不得编造。
- 提交日期

## 第一部分 股权投资项目方案

### 一、项目背景
标的公司一句话定位（成立时间、技术/业务方向、行业地位、总部及研发布局）、与本机构现有业务关系（是否有结算/信贷合作）、历史融资轮次与累计融资额、上市计划意向。这段是全文唯一一处"电梯陈述"，后续章节展开细节，不要在这里堆砌数据。

### 二、尽调的组织与实施情况
列举实际做过的尽调动作，通常包括：
- 依据的三方尽调（如有联合投资方委托的会计师/律师事务所开展的财务/法律尽调，注明机构名称）
- 书面材料审阅（商业计划书、审计报告等）
- 现场调研与高管访谈
- 公开信息核查（国家企业信用信息公示系统、融安E信/征信系统等，确认经营状态、失信名单、税务异常、行政处罚）
- 访谈主要投资人、下游客户、行业专家
- 盈利与项目收益测算的依据

只写实际做过的动作；没有做的尽调环节不要虚构。

### 项目方案
#### 方案概述
本次投资通过什么主体/工具参与（基金/SPV名称）、出资上限、投资期限安排、退出逻辑一句话概述。

#### 基本交易要素
固定字段清单（缺项写"未披露"）：投资标的、投资规模（含机构自有资金部分）、投前估值、资金来源、投资期限、资金用途、持股比例（及计算依据）、投资收益来源、投资退出方式。

#### 基金方案
如通过基金/合伙企业参与，覆盖：基金名称、组织形式、注册地、基金规模、存续期（投资期/退出期/可延长安排）、GP结构（单GP/双GP及各自角色）、LP结构及认缴比例、投资方向限制（如"科技领域创新型中小企业份额不低于X%"）、投资决策机制（决策委员会构成、表决规则）、管理费/执行事务费费率、退出方式、收益分配瀑布（分配顺序、优先回报率、超额收益分成比例）、需合伙人会议/决策委员会审议的重大事项清单。

#### 本轮融资安排及上市计划
本轮估值及与其他领投方估值的比较、主要保障条款（分红权、优先认购权、优先购买权、随售权、优先清算权、反稀释、信息权等，按实际协议列出，不要泛泛列举所有可能条款）、回购触发情形清单（逐条列出，通常包括：未如期IPO、关键里程碑未达成、实控人变更、未经同意变更主业、股权纠纷、其他投资人行使回购权等）、回购主体与回购价格公式（本金+年化利率+已宣布未分配红利）、募集资金用途、上市计划时间表。

## 第二部分 投资标的企业情况

### 一、基本情况
#### 公司简介
固定信息表（名称、所属行业、注册地址、经营期限、法定代表人、成立时间、注册资本、实收资本、企业性质、营业执照号、主要产品及服务），随后：
- 公司股权结构表（股东名称+持股比例，合计100%）
- 历次融资表（轮次、融资时间、融资额、投后估值、投资人），及最新一轮估值/交割条件的特别说明

#### 公司治理结构
- 公司治理层级（股东会/董事会/监事会的决策权限、普通决议与特别决议表决门槛）、董事会构成（各股东提名董事人数、董事长产生方式、特殊事项的一票否决/共同决策安排）
- 组织架构（部门设置及职责表）
- 核心团队（创始人及高管、技术合伙人逐人列出：姓名、职务、教育背景、过往任职、可验证的具体成就——不写"经验丰富"这类空话，要具体到机构名+年限+可验证事实；技术/学术背景可通过公开检索交叉验证）

#### 子公司及参股公司情况
控股子公司/参股公司表：公司名称、主营业务、注册资本、持股比例、资产、负债、营业收入、净利润、银行融资情况。

### 二、企业经营情况
主营产品/技术路线及演进（代表性产品逐代对比：性能参数、量产/流片进展、目标应用场景）、近期重大商业化进展（中标、量产、战略合作，注明数量/金额/时点）、客户结构。所有数字必须来自提供材料或可核实的公开信息（如公司官网、公开报道、招投标公告）。

### 三、财务情况
说明审计情况（哪几年审计报告、审计机构、审计意见类型，是否含未经审计的最新报表）。随后按三张报表分别展开，每张表用近三年+最新一期数据、含同比变化列：

- **资产负债表趋势表**：流动资产、非流动资产、资产总计；流动负债、非流动负债、负债合计；所有者权益。表后按（一）资产负债分析 → 1.合并资产（流动/非流动资产逐科目变动说明+占比表）→ 2.合并负债（逐科目变动说明）→ 资产负债率变化及杠杆判断。
- **利润表+现金流量表趋势表**：营业总收入、营业总成本（细分销售/管理/研发/财务费用）、营业利润、利润总额、净利润；经营/投资/筹资活动现金流量净额、期末现金及现金等价物余额。表后按 1.合并盈利能力（收入增长率、毛利率、分产品/分客户收入与毛利明细表）→ 2.合并现金流量（三类现金流变动原因）展开。

每一段变动说明遵循固定句式："XX科目金额为X万元，较上年同期增长/下降X%，主要是因为……"——原因必须来自访谈或材料，不能凭空归因。

### 四、资信情况
- （一）行内评级情况：本机构内部评级结果、近期授信额度核定情况及使用条件
- （二）融资情况：外部征信查询结果——银行融资台账表（贷款银行、借款类型、日期、利率、金额、担保方式）
- （三）信用情况及重大诉讼情况：法院被执行人查询、企业信用信息公示系统查询、内部征信系统查询结果（是否有诉讼、失信记录）
- （四）对外担保：人行征信查询的对外担保情况

这一整节几乎全部依赖征信/工商查询结果和内部系统数据——没有查询到的项目要写"未查询到相关信息"而不是省略小节。

## 第三部分 行业分析与公司优势分析

### 一、[核心技术/行业]发展情况
技术背景与原理、技术分类、上下游/介质或路线选择、发展趋势、应用领域、行业挑战与难点。这一节是纯行业研究内容，来源应为公开可查的行业报告、技术论文、专家访谈——标注信息来源类型（如"据XX访谈"、"据公开资料"）。

### 二、公司竞争优势
逐条展开（每条一个小标题+一段论证），常见维度：底层技术/专利壁垒、核心团队稀缺性、目标市场清晰度与产品优势（附第三方市场预测数据源）、产业资本/战略客户赋能、供应链/国产替代战略意义。每条优势都要有具体依据（数字、专利、客户名、第三方预测机构名），不写空泛评价。

## 第四部分 同业可比分析

### 国内外市场同行业企业
先给出竞争图谱/分类逻辑（按技术路线或目标场景分层），再逐一介绍主要竞争对手：成立时间、所在地、技术路线、产品进展、收入规模（如可查）、融资历史、管理层背景——全部标注来源（公开报道、企业官网、专家访谈）。覆盖境内外、大厂和创业公司两类可比对象。

### 经营情况对比
产品参数对比表（按下游场景分组，如有多个场景可分别列表：算力、功耗、能效比、带宽、制程、售价等）、以及传统架构/替代技术厂商的正面对比（附对方最新的收入、毛利率、净利润等公开披露数据）。

## 第五部分 投资收益情况分析

### 一、盈利预测
说明预测的基础（管理层提供的预测 + 项目组的谨慎性折扣，注明折扣比例及理由）、已签约合同金额与收入确认节奏的差异说明、未来收入结构预期（主要客户/渠道）。所有预测数字必须标注"基于管理层假设，项目组按X%折扣测算"这类归因，不得当作既成事实陈述。

### 二、投资回报分析
可比上市公司估值倍数表（代码、简称、总市值、最新年度营业收入/净利润、毛利率、净利率、PE(TTM)、PS(TTM)，取平均值作为退出倍数假设）——必须使用真实可查的上市公司数据。随后给出至少两组情景假设（如"中性假设"/"保守假设"）：投资金额、投后估值、各阶段稀释比例、退出时点、预测退出年度营业收入、采用的退出倍数、测算IRR。每组情景都要列出假设前提，不要只给结论数字。若IPO不及预期，说明其他退出路径（并购、卖老股、回购）。

## 第六部分 上市退出可行性、风险分析及防控措施

### 上市退出可行性分析
上市板块选择及备选方案、回购兜底条款重述（与第一部分交易要素一致）。

### 二、本笔业务的主要风险及措施
编号风险清单，每条固定结构：
1. **风险标题**（一句话概括）
   风险描述段落——具体到数字、事件、协议条款，来自访谈或材料，不得泛泛而谈。
   **应对措施：**具体、可执行的缓释动作段落——不能是"密切关注"这类空话单独出现，要说明关注什么指标、要求公司做什么承诺/陈述保证、投后阶段如何跟进。

常见风险类别（按需覆盖，不是每单都全部适用）：收入结构/商业化阶段性风险、业务方向调整/竞争格局风险、持续融资需求与资金缺口、跨境合规风险（出口管制、外资安全审查、创始人国籍等）、客户集中度与关联交易风险、地方政府扶持协议/业绩承诺相关的或有负债、股权激励/期权相关的历史合规问题、核心人员对外任职/知识产权归属风险。

## 第七部分 调查结论性意见

一段综合判断：公司的核心竞争优势与商业化路径（呼应第三部分）、当前阶段的核心不确定性（呼应第五、六部分的风险），随后是**明确的投资建议**——是否建议投资、通过什么主体/基金、投前估值上限、出资金额上限（含自有资金部分），与第一部分"基本交易要素"保持数字一致。这是决策意见，不是摘要复述，必须给出可执行的结论。

## 无来源占位符使用规范

以下情形一律使用固定占位表述，禁止编造具体数值/姓名/日期去填补：
- 数据缺失："未披露"
- 材料未提供但应有此项："资料未提供，待补充"
- 需要进一步核实的推测或管理层单方陈述："据公司反馈，需进一步核实"
